GFmotor API 測試介面

通用

API 根路徑: /api/v1

使用者 (Users)

建立測試用使用者 (POST /users/)

此功能僅供測試，會自動產生假的 Google ID。可選填車籍資料。

基本資料

車籍資料 (選填)

讀取使用者列表 (GET /users/)
讀取單一使用者 (GET /users/{id})
更新使用者資訊 (PUT /users/{id})

基本資料 (選填)

新增車籍資料 (選填)

此處只會「新增」車輛，不會修改舊有車籍。

刪除使用者 (DELETE /users/{id})

車籍管理 (Motors)

更新車籍資料 (PUT /motors/{id})

（只需填寫要更新的欄位）

刪除車籍 (DELETE /motors/{id})

商品 (Products)

讀取商品列表 (GET /products/)
讀取單一商品 (GET /products/{id})
建立新商品 (POST /products/)
更新商品資訊 (PUT /products/{id})

（只需填寫要更新的欄位）

刪除商品 (DELETE /products/{id})

預約 (Bookings)

建立新預約 (POST /bookings/)

查詢結果會顯示在頁面下方，請將查到的 Google ID 和車籍 ID 複製到下方的欄位中。

步驟 2: 填寫預約資料並建立

讀取預約列表 (GET /bookings/)
讀取單一預約 (GET /bookings/{id})
更新預約資訊 (PUT /bookings/{id})

工單 (Work Orders)

建立新工單 (POST /work-orders/)

新增零件項目

目前工單項目:

讀取工單列表 (GET /work-orders/)
讀取單一工單 (GET /work-orders/{id})
更新工單資訊 (PUT /work-orders/{id})
