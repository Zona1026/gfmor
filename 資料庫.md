# 資料庫架構說明

本文件說明 `gfmotor` 資料庫中所有資料表(table)的結構與用途。

---

## 1. `admins` - 管理員資料表

- **功能**: 儲存後台管理員的帳號資訊與權限。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `帳號` | int | PRIMARY KEY, NOT NULL, UNIQUE | 管理員的唯一識別帳號。 |
  | `密碼` | varchar(45) | NOT NULL | 管理員登入密碼 (應加密儲存)。 |
  | `名稱` | varchar(45) | NOT NULL | 管理員顯示名稱。 |
  | `權限` | enum('最高級','管理層','一般') | DEFAULT '一般' | 帳號權限等級。 |

---

## 2. `users` - 使用者(客戶)資料表

- **功能**: 儲存客戶的基本資料，使用 Google ID 作為主要識別碼。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `Google ID` | varchar(255) | PRIMARY KEY, NOT NULL | 使用者的 Google 帳號唯一識別碼。 |
  | `車主姓名` | varchar(10) | NOT NULL | 客戶的真實姓名。 |
  | `電話` | varchar(10) | NULL | 客戶的聯絡電話。 |
  | `email` | varchar(100) | NOT NULL, UNIQUE | 客戶的電子郵件。 |
  | `類別` | enum('MEMBER','ADMIN') | DEFAULT 'MEMBER' | 使用者類別 (一般會員或管理員)。 |
  | `加入時間` | datetime | DEFAULT CURRENT_TIMESTAMP | 帳號建立時間。 |
  | `會員等級` | varchar(45) | NULL | 客戶的會員等級。 |
  | `累積消費` | int | NULL | 客戶的歷史消費總額。 |
  | `車牌` | varchar(45) | NULL | 客戶的主要車牌 (可能有多台車)。 |

---

## 3. `motor` - 車籍資料表

- **功能**: 儲存客戶擁有的車輛資訊，一位客戶可以有多台車。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `ID` | int | PRIMARY KEY, AUTO_INCREMENT | 車籍資料的唯一流水號。 |
  | `Google ID` | varchar(255) | NOT NULL, FOREIGN KEY | 關聯到 `users` 資料表，表示車主。 |
  | `車牌` | varchar(45) | NOT NULL | 車輛的車牌號碼。 |
  | `引擎號碼` | varchar(45) | NULL | 車輛的引擎號碼。 |
  | `車種` | varchar(45) | NULL | 車輛的型號或名稱 (例如: JETS)。 |

---

## 4. `bookings` - 預約紀錄資料表

- **功能**: 儲存客戶的服務預約紀錄。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `預約單號` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 預約單的唯一流水號。 |
  | `Google ID` | varchar(255) | NOT NULL, FOREIGN KEY | 關聯到 `users` 資料表，表示預約的客戶。 |
  | `車籍ID` | int | NOT NULL, FOREIGN KEY | 關聯到 `motor` 資料表，表示預約的車輛。 |
  | `預約時間` | datetime | NOT NULL | 客戶預約的到店時間。 |
  | `類別` | enum('維修','保養','諮詢') | NOT NULL | 預約的服務類型。 |
  | `預約單成立時間` | datetime | DEFAULT CURRENT_TIMESTAMP | 這筆預約紀錄的建立時間。 |
  | `狀態` | enum('預約中','預約取消','已超時','已結案','後台開放時間') | NOT NULL | 目前預約的狀態。 |
  | `備註` | varchar(100) | NULL | 客戶在預約時留下的備註。 |

---

## 5. `consumptions` - 消費紀錄資料表

- **功能**: 簡單記錄客戶的每一筆消費。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `id` | int | PRIMARY KEY, AUTO_INCREMENT | 消費紀錄的唯一流水號。 |
  | `Google ID` | varchar(255) | NOT NULL, FOREIGN KEY | 關聯到 `users` 資料表，表示消費者。 |
  | `消費金額` | int | NOT NULL | 該筆消費的總金額。 |
  | `消費項目` | varchar(100) | NOT NULL | 該筆消費的項目摘要。 |
  | `消費時間` | datetime | NULL | 消費發生的時間。 |

---

## 6. `portfolio_items` - 作品集項目資料表

- **功能**: 儲存店家完成的改裝或維修作品，用於網站上展示。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `id` | int | PRIMARY KEY, AUTO_INCREMENT | 作品的唯一流水號。 |
  | `標題` | varchar(100) | NOT NULL | 作品的標題。 |
  | `描述` | varchar(500) | NULL | 作品的詳細描述。 |
  | `分類` | varchar(50) | NOT NULL | 作品的分類 (例如: 懸吊系統, 引擎升級)。 |
  | `圖片路徑` | varchar(255) | NOT NULL | 作品照片的儲存路徑或URL。 |
  | `上傳時間` | datetime | NULL | 作品上傳的時間。 |

---

## 7. `products` - 庫存商品資料表



- **功能**: 管理店內的所有可販售商品庫存。

- **欄位**:

  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:--- |:--- |:--- |:--- |
  | `id` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 商品的唯一流水號。 |
  | `name` | varchar(100) | NOT NULL | 品名。 |
  | `description` | text | NULL | 商品的詳細描述。 |
  | `price` | int unsigned | NOT NULL | 價格。 |
  | `stock` | int unsigned | NOT NULL, DEFAULT 0 | 庫存數量。 |
  | `category` | varchar(50) | NULL | 商品分類。 |
  | `created_at` | datetime | DEFAULT CURRENT_TIMESTAMP | 商品建立時間。 |



---



## 8. `work_orders` - 工單資料表



- **功能**: 追蹤每一項維修或改裝工作的詳細記錄，對應到一筆預約。

- **欄位**:

  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:--- |:--- |:--- |:--- |
  | `id` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 工單的唯一流水號。 |
  | `booking_id` | int unsigned | NOT NULL, FOREIGN KEY | 關聯到 `bookings` 資料表。 |
  | `status` | enum('待處理', '處理中', '待付款', '已完成', '取消') | NOT NULL, DEFAULT '待處理' | 工單目前的處理狀態。 |
  | `total_amount`| int unsigned | NOT NULL, DEFAULT 0 | 此工單的總金額。 |
  | `notes` | text | NULL | 關於此工單的內部備註。 |
  | `created_at` | datetime | DEFAULT CURRENT_TIMESTAMP | 工單建立時間。 |
  | `completed_at`| datetime | NULL | 工單完成時間。 |



---



## 9. `work_order_items` - 工單項目詳情資料表



- **功能**: 記錄每一張工單具體使用了哪些商品以及數量。

- **欄位**:

  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:--- |:--- |:--- |:--- |
  | `id` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 流水號。 |
  | `work_order_id` | int unsigned | NOT NULL, FOREIGN KEY | 關聯到 `work_orders` 資料表。 |
  | `product_id` | int unsigned | NOT NULL, FOREIGN KEY | 關聯到 `products` 資料表。 |
  | `quantity` | int unsigned | NOT NULL | 使用的商品數量。 |
  | `unit_price` | int unsigned | NOT NULL | 執行當下的商品單價。 |

---

## 10. `orders` - 客戶線上訂單資料表

- **功能**: 儲存客戶透過線上商城所下的訂單。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `id` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 訂單的唯一流水號。 |
  | `google_id` | varchar(255) | NOT NULL, FOREIGN KEY | 關聯到 `users` 資料表，表示下單的客戶。 |
  | `status` | enum('待處理', '處理中', '已出貨', '已完成', '已取消') | NOT NULL, DEFAULT '待處理' | 訂單目前的處理狀態。 |
  | `total_amount`| int unsigned | NOT NULL | 此訂單的總金額。 |
  | `recipient_name`| varchar(50) | NOT NULL | 收件人姓名。 |
  | `recipient_phone`| varchar(20) | NOT NULL | 收件人電話。 |
  | `shipping_address`| varchar(255) | NOT NULL | 運送地址。 |
  | `notes` | text | NULL | 客戶留下的訂單備註。 |
  | `created_at` | datetime | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 訂單建立時間。 |
  | `updated_at` | datetime | NOT NULL, DEFAULT CURRENT_TIMESTAMP, ON UPDATE CURRENT_TIMESTAMP | 訂單最後更新時間。 |

---

## 11. `order_items` - 訂單項目詳情資料表

- **功能**: 記錄每一張訂單具體購買了哪些商品、數量及當時的單價。
- **欄位**:
  | 欄位名稱 | 資料型態 | 欄位設定 | 說明 |
  |:---|:---|:---|:---|
  | `id` | int unsigned | PRIMARY KEY, AUTO_INCREMENT | 流水號。 |
  | `order_id` | int unsigned | NOT NULL, FOREIGN KEY | 關聯到 `orders` 資料表。 |
  | `product_id` | int unsigned | NOT NULL, FOREIGN KEY | 關聯到 `products` 資料表。 |
  | `quantity` | int unsigned | NOT NULL | 購買的商品數量。 |
  | `unit_price` | int unsigned | NOT NULL | 購買當下的商品單價。 |
