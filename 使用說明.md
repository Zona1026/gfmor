# 使用說明

本文件將指導您如何設定並啟動本專案的開發環境。

---

## 專案結構

- **`/` (根目錄)**: 包含後端 FastAPI 應用程式及相關設定。
- **`/frontend`**: 包含前端 Vue.js 應用程式。

---

## 後端 (FastAPI)

### 1. 安裝依賴

請確認您已安裝 Python。進入專案根目錄，建議先建立並啟用虛擬環境。

```bash
# 進入專案根目錄
cd d:\Python\GFmoter

# 安裝所有必要的 Python 套件
pip install -r requirements.txt
```

### 2. 環境變數

複製 `.env.sample` 檔案並重新命名為 `.env`。

```bash
copy .env.sample .env
```

請根據您的本地端資料庫設定，修改 `.env` 檔案中的 `DATABASE_URL`。

### 3. 啟動伺服器

在專案根目錄下執行以下指令：

```bash
uvicorn main:app --reload
```

伺服器預設會啟動於 `http://127.0.0.1:8000`。

---

## 前端 (Vue.js)

### 1. 安裝依賴

請確認您已安裝 Node.js 與 npm。進入前端專案目錄。

```bash
# 進入前端專案目錄
cd frontend
# 或者
cd d:\Python\GFmoter\frontend

# 安裝所有必要的 Node.js 套件
npm install
```

### 2. 啟動開發伺服器

在 `frontend` 目錄下執行以下指令：

```bash
npm run dev
```

前端開發伺服器預設會啟動於 `http://localhost:5173`。您可以開啟瀏覽器進入此網址來查看網站。
